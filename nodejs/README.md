# Node.js (TypeScript) Playbook

**Purpose:** a self-contained playbook for Node.js backend projects written in TypeScript. Use this to get a new project running quickly, capture learnings, and standardize templates.

## TL;DR Quick commands

```bash
# clone repo and enter nodejs folder
git clone <repo-url>
cd nodejs/examples/minimal-app
# install
pnpm install # or npm install
# run in dev
pnpm dev # or npm run dev
# build
pnpm build
# run production
pnpm start
